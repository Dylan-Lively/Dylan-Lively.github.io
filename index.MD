## Trace

A gesture-controlled 3-DOF robotic arm built for my engineering course.

A Raspberry Pi tracks hand position in real time using MediaPipe and streams movement commands over serial to an Arduino, which solves inverse kinematics and drives three stepper motors.

**Gestures:**
- Victory — relative tracking, arm follows hand movement
- Open palm — absolute snap, tip locks to hand position
- ILY — mirror mode, arm moves opposite to hand
- Anything else — hold position

**Built with:** Python · MediaPipe · OpenCV · Arduino · AccelStepper
